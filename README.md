# Daidex Beta Version:

## In this beta version new functionalities will be tested. Specially those related to ZeroEx library (`0x.js`) and Web3 (`web3.js`)

### Currently this repo uses the following versions
* `0x.js@0.38.6`
* `bignumber@5.0.0`
* `web3@1.0.0`

## How to install
```sh
$ git clone <repo>

$ npm install

$ npm run build

$ npm start
```
Go to *localhost:3000*
